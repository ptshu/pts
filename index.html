MarkDown使用心得

留言

MarkDown语法：

标题：

在md格式下使用”#”来标记标题 ，有N个#时表示N级标题,N最大为6。（#后需要接一个空格）

段落说明：

段落直接编写文字，没有特殊格式。
段落的换行是使用两个以上空格加上回车。
段落后面使用一个空行来表示重新开始一个段落。

hexo常用命令

hexo new “name” # 新建文章
hexo new page “name” # 新建页面
hexo g # 生成页面
hexo d # 部署
hexo g -d # 生成页面并部署
hexo s # 本地预览
hexo clean # 清除缓存和已生成的静态文件
hexo help # 帮助
