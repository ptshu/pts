valine错误

2024年06月02日
hexo博客
留言

valine提示以下错误：
Code 504: The app is archived, please restore in console before use.


原因是：valine超过30天没有使用被停止，存储服务数据已归档（连续30天无API请求）

解决办法：点上图中的激活，然后等10-30分钟后就激活成功。
